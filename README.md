## 🚀 Overview

**FiveM Utility** is a powerful Windows desktop application designed to streamline your FiveM experience. Whether you're a regular player looking to clean up your cache and improve performance, or a power user managing multiple `.rpf` mods and visual settings—this tool puts everything you need into a modern, fast, and easy-to-use interface.

Built with **C# and WinUI 3**, it feels native to Windows 11 while providing deep integration with your local FiveM directory.

## ✨ Features

- 🧹 **Advanced Cache Cleaner**
  - Safely clear temporary files, server cache, and crash dumps.
  - Choose between basic cleanup (keeps your game settings) or deep cleaning.
  - Fix common connection and asset loading issues instantly.

- 📦 **Mods Manager**
  - View all installed `.rpf` files and plugins at a glance.
  - Easily toggle mods on and off without deleting files.
  - Identify conflicting visual mods or outdated plugins.

- 🔍 **RPF Explorer (Built-in)**
  - Browse the internal structure of any `.rpf` archive directly.
  - Auto-parse and display formatted `assembly.xml` mod information (Name, Author, Install Targets).
  - Search for specific files inside archives and export them to your disk.
  - No need to extract GBs of data just to check what's inside a mod!

- 📊 **Dashboard & Backups**
  - Monitor your FiveM installation size and disk usage.
  - One-click backups of your `mods` folder and `citizen/common/data` configurations before making risky changes.

## 📥 Download & Installation

1. Go to the [Releases](https://github.com/[YOUR-USERNAME]/[YOUR-REPO]/releases) page.
2. Download the latest `FiveMUtility_Setup_x.x.x.exe`.
3. Run the installer and follow the instructions.
4. Launch **FiveM Utility** from your Start Menu!

> **Note:** The application will automatically attempt to locate your FiveM installation. If your FiveM is installed in a custom location, you can set the path manually in the **Settings** tab.
